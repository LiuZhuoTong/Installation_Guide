ComfyUI容器化安装手册（验证无误）

一、安装docker、nvidia-runtime
租用捷智云主机（https://www.gogpu.cn），选择自带 docker 和 nvidia-runtime 的系统镜像，可以更加方便快捷
1、docker 安装验证：docker --version
2、nvidia-runtime 安装验证：ls -l /usr/bin/nvidia-container-runtime*
（会在/usr/bin/下找到相关文件）

二、快速测试 Docker 容器能否正常访问宿主机的 NVIDIA GPU
docker run --rm --gpus all nvidia/cuda:12.2.0-base-ubuntu22.04 nvidia-smi

三、创建工作目录
1、cd ~
2、mkdir -p comfyui

四、拉取镜像
docker pull m5dcw6cxn4ifp7yjft.xuanyuan.run/yanwk/comfyui-boot:cu128-slim
（m5dcw6cxn4ifp7yjft为个人私钥，需要个人妥善保存避免暴露）

五、镜像验证
docker images | grep comfyui-boot

六、构建工作目录
cd ~/comfyui
mkdir -p \
  storage \
  storage-models/models \
  storage-models/hf-hub \
  storage-models/torch-hub \
  storage-user/input \
  storage-user/output \
  storage-user/workflows

七、启动容器
1、cd ~/comfyui
2、docker run -d -it --rm \
  --name comfyui \
  --gpus all \
  -p 8188:8188 \
  -v "$(pwd)"/storage:/root \
  -v "$(pwd)"/storage-models/models:/root/ComfyUI/models \
  -v "$(pwd)"/storage-models/hf-hub:/root/.cache/huggingface/hub \
  -v "$(pwd)"/storage-models/torch-hub:/root/.cache/torch/hub \
  -v "$(pwd)"/storage-user/input:/root/ComfyUI/input \
  -v "$(pwd)"/storage-user/output:/root/ComfyUI/output \
  -v "$(pwd)"/storage-user/workflows:/root/ComfyUI/user/default/workflows \
  m5dcw6cxn4ifp7yjft.xuanyuan.run/yanwk/comfyui-boot:cu128-slim

八、端口暴露
1、在云服务器上暴露内网端口8188，并获取对应的外网端口为31134。
2、通过hostIp:31134访问comfyui界面服务
