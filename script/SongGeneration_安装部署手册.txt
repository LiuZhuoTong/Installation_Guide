SongGeneration安装步骤

一、安装docker（如果安装过则体跳过）
验证安装：docker --version

二、安装nvidia-docker（如果安装过则跳过）
验证安装过：docker run --rm --gpus all nvidia/cuda:11.0-base nvidia-smi

三、拉取容器
容器中只含运行环境不含模型源码和权重
docker pull juhayna/song-generation-levo:hf0613

三、代码下载
1、cd ~
2、下载源代码：git clone https://github.com/tencent-ailab/SongGeneration.git
3、如果GitHub慢，用GitCode：git clone https://gitcode.com/tencent_hunyuan/SongGeneration.git
4、cd SongGeneration
5、mkdir output

四、模型运行时下载（基础权重 + 三方依赖）
1、pip install huggingface-hub
2、cd ~/SongGeneration
3、huggingface-cli download lglg666/SongGeneration-Runtime --local-dir ./runtime
4、mv runtime/ckpt ckpt
5、mv runtime/third_party third_party

五、模型权重下载
1、cd ~/SongGeneration
2、mkdir -p models
3、cd models
4、huggingface-cli download lglg666/SongGeneration-base-full --local-dir ./songgeneration_base_full

六、测试
1、docker run -it --gpus all -v ~/SongGeneration:/workspace --network=host juhayna/song-generation-levo:hf0613 /bin/bash
2、cd /workspace/SongGeneration
3、sh generate.sh models/songgeneration_base_full sample/lyrics.jsonl output

七、后台运行
1、启动容器：docker run -d --gpus all -v ~/SongGeneration:/workspace --network=host juhayna/song-generation-levo:hf0613 bash -c "cd /workspace && sh tools/gradio/run.sh models/SongGeneration-base-full && tail -f /dev/null"
2、界面访问：http://您的服务器IP:8081

八、高级选项
1、低显存模式：sh generate.sh ckpt_path lyrics.jsonl output_path --low_mem
2、禁用flash_attention：sh generate.sh ckpt_path lyrics.jsonl output_path --not_use_flash_attn
3、生成纯音乐：sh generate.sh ckpt_path lyrics.jsonl output_path --bgm
4、生成纯人声：sh generate.sh ckpt_path lyrics.jsonl output_path --vocal
5、分离人声的伴奏：sh generate.sh ckpt_path lyrics.jsonl output_path --separate
6、界面操作：启动sh tools/gradio/run.sh songgeneration_base_full

九、输入文件格式说明
1、歌词格式示例（lyrics.jsonl）
{
  "idx": "song_001",
  "gt_lyric": "[intro-short] ; [verse] These faded memories of us. I can't erase the tears you cried before ; [chorus] Like a fool begs for supper. I find myself waiting for her",
  "descriptions": "female, pop, sad, piano",
  "prompt_audio_path": "/path/to/reference.wav"
}

2、可用的结构标签
  2.1、纯器乐段落（不含歌词）：
    [intro-short]、[intro-medium]：前奏（短/中）
    [inst-short]、[inst-medium]：间奏（短/中）
    [outro-short]、[outro-medium]：尾奏（短/中）
  2.2、需要歌词的段落：
    [verse]：主歌
    [chorus]：副歌
    [bridge]：桥段

3、描述字段选项
性别：male, female
音色：dark, bright, soft
风格：pop, jazz, rock, R&B, Dance, Folk
情感：sad, energetic, romantic
乐器：piano, drums, guitar
BPM：the bpm is 120

七、注意事项
1、不要同时提供 prompt_audio_path 和 descriptions，否则可能导致生成质量下降
2、参考音频只使用前10秒
3、建议使用歌曲的副歌部分作为参考音频
4、如果遇到显存不足问题，请使用 --low_mem 参数
